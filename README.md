# TaskFlow — Premium Productivity App

A stunning, production-ready productivity application built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- ✅ Task management with priorities (High / Medium / Low)
- 📅 Due dates with live countdown timers
- 🎯 Focus mode (today's + high-priority tasks)
- 📆 Weekly schedule view with workload capacity bars
- 🗓 Monthly calendar view with day detail panel
- 🚀 Preparation planner (auto-calculates h/day to meet deadlines)
- 🔥 Completion streak tracker
- 🎉 Confetti when all tasks are completed
- 🌙 Dark / Light mode with animated toggle
- 🔍 Search and filter tasks
- ↕️ Drag-and-drop reordering (dnd-kit)
- 💾 LocalStorage persistence
- 📱 Fully responsive (mobile-first)

## Tech Stack

- React 18 + TypeScript
- Framer Motion (animations)
- Tailwind CSS (styling + dark mode)
- @dnd-kit/core + @dnd-kit/sortable (drag and drop)
- date-fns (date utilities)
- canvas-confetti (celebration)
- lucide-react (icons)
- Vite (build tool)

